# Estrutura for em JavaScript

Estrutura for em JavaScript

Exemplos:

```js
for(let i = 0; i <= 10; i++) {
    console.log(`i = ${i}`);
};
```

Ler um array com for:

```js
const notas = [6.1, 7.2, 5.3, 8.9, 5.8];
for(let i = 0; i <= notas.length; i++) {
    console.log(`nota = ${notas[i]}`);
};
```
